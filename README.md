# BiliArm

BiliArm 是一款用于优化哔哩哔哩网页端观看体验的浏览器插件。它可以帮助用户自动处理一些常见的重复操作，例如关闭弹幕、自动播放、切换显示模式、关灯观看以及隐藏干扰元素等。

项目地址：[https://github.com/newAres/BiliArm](https://github.com/newAres/BiliArm)

## 主要功能

- 插件总开关：一键启用或停用所有增强功能。
- 默认关闭弹幕：打开视频页面后自动关闭弹幕。
- 自动播放：进入播放页后自动开始播放视频。
- 播放结束退出全屏：视频播放结束后自动退出全屏或网页全屏。
- 屏蔽底部弹幕：减少底部弹幕遮挡字幕的情况。
- 默认关灯：进入播放页后自动开启沉浸观看效果。
- 滚动自动开关灯：根据页面滚动行为自动调整关灯状态，适合宽屏观看。
- 默认显示模式：支持正常、宽屏、网页全屏等显示模式。
- 去除大屏轮播图：隐藏影响观看或浏览体验的轮播内容。
- 换一换保留旧列表：首页点击换一换后保留原视频列表。

## 适用人群

如果你经常在哔哩哔哩网页端观看视频，并且希望减少每次进入视频页后的重复点击，BiliArm 会比较适合你。

它尤其适合以下场景：

- 经常使用宽屏或网页全屏观看视频。
- 希望打开视频后自动关闭弹幕。
- 不喜欢页面轮播图等干扰元素。
- 希望获得更专注的观看体验。

## 使用方式

当前开发版可以按以下方式加载：

1. 下载或克隆本项目。
2. 打开浏览器扩展管理页面。
3. 启用开发者模式。
4. 选择“加载已解压的扩展程序”。
5. 选择本项目根目录。
6. 打开哔哩哔哩网页端，根据需要调整插件设置。

当前版本仍处于开发阶段，部分依赖页面按钮或选择器的功能可能会随着 B 站页面结构变化而需要适配。

## 配置说明

BiliArm 提供独立设置页面，用户可以按需开启或关闭每项功能。总开关关闭后，插件不会改变哔哩哔哩页面的默认行为，但会保留用户已经选择的配置。

默认显示模式计划支持：

- 正常模式
- 宽屏模式
- 网页全屏模式

## 权限说明

BiliArm 会尽量使用最少的浏览器权限。当前设计中主要需要：

- 存储权限：保存用户设置。
- 哔哩哔哩页面访问权限：在目标页面执行观看体验优化。

插件不会用于下载视频、绕过会员限制、绕过登录限制或修改用户账号数据。
